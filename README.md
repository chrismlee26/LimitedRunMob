# Limited Run Mobile App

v0.1.0

- Built in React-Native, Expo, Navigation

- Design: Figma

## Project

- This project is the mobile front-end to display information provided by the (web-admin-panel project)[https://github.com/chrismlee26/limitedrunweb]

- Product development isn't easy. v0.2.0 web is a set of tools that will help makers organize thoughts to make better decisions. These plans are shared to buyers who can view on this mobile application.

- Real Time Database will be updated in the next release

## Dependencies & Run

Type <npm install> for dependencies

Type <yarn start> to view application in Expo. Run in xCode iOS Simulator. Currently the application may or may not only work on iOS.
